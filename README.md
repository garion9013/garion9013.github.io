1. Install packages
```bash
# install packages
$ gem install --user-install bundler jekyll rouge jekyll-paginate
```

2. Setup environment
``` bash
# set environment (OSX)
export PATH=$HOME/.gems/ruby/2.5.0/bin:$PATH
```

3. Launch server
```bash
$ jekyll serve
# Then, check it on your web browser (localhost:4000)
```
